Installation
============

To use ``Makefile`` you need to have ``make`` installed on your system.   

.. tabs::

    .. tab:: Ubuntu

        .. include:: installation_build_status_badge.rst
        .. code-block:: bash

            sudo apt-get install make

    .. tab:: RedHat

        .. warning:: GitHub Actions are not available for RedHat based systems.
        .. code-block:: bash

            sudo yum install make

    .. tab:: MacOS

        .. include:: installation_build_status_badge.rst
        .. code-block:: bash

            brew install make

    .. tab:: Windows

        .. include:: installation_build_status_badge.rst
        .. code-block:: bash

            choco install make
