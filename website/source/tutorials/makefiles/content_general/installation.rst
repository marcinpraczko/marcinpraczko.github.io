Installation
============

To use ``Makefile`` you need to have ``make`` installed on your system.   

.. tabs::

    .. tab:: Ubuntu

        .. code-block:: bash

            sudo apt-get install make

    .. tab:: RedHat

            .. code-block:: bash

                sudo yum install make

    .. tab:: MacOS

        .. code-block:: bash

            brew install make

    .. tab:: Windows

        .. code-block:: bash

            choco install make
